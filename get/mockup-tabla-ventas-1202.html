Notas de implementación:
  • Columna fija izquierda (Carrito): Usa PrimeVue DataTable con frozen + scrollable, igual que TableClients.vue de Vista 360.
  • Columna fija derecha (Acciones): alignFrozen="right" + frozen.
  • Columna Asesor: Solo visible en Gestión de Ventas, oculta en Mis Ventas (prop configurable).
  • Columna Pedido: Si pedido_id_convertido > 0 muestra #PED + badge estado pago. Si es 0/null muestra "— Sin pedido".
  • Link "Ir a checkout": Solo visible si el pedido está pendiente de pago. Lleva al checkout del pedido original.
  • Tabs: "WhatsApp" reemplaza "Mis ventas". "Archivados" es nuevo (usa campo carrito_asistido_archivado).
  • El #carrito_id es un link que abre el detalle del carrito (misma función que el botón "Ver").
